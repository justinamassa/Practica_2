# Instalación 
1. Clonar el repositorio: git clone https://github.com/justinamassa/Practica_2.git

2. Crear el entorno virtual: python -m venv venv

3. Activar el entorno virtual: .\venv\Scripts\activate

4. Instalar las dependencias: pip install -r requirements.txt

# Ejecución 
Abrir VS Code, ir a la carpeta  `notebooks/` y ejecutar el notebook del ejercicio deseado con **Run All**
